# 术语表

此文件用于统一中文翻译口径，避免同一术语在不同文档中出现多个译法。

## 建议术语

- TrajectoryRL - TrajectoryRL
- miner - 矿工
- validator - 验证者
- pack - 策略包
- commitment - 链上承诺
- evaluation - 评估
- ClawBench - ClawBench
- policy pack - 策略包
- prompt - 提示词
- judge - 裁判模型

## 维护规则

- 新术语先补到这里，再开始翻译正文。
- 如果某个英文词保留不译，统一在这里说明原因。
- 术语表优先保证一致性，其次才是字面准确。
